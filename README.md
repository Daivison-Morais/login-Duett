## Para rodar localmente

Clone o repositório, na raíz do projeto rode:

```bash
$ npm install
```
```bash
$ npm run build
```

```bash
$ npm run start
```

modifique a BASE_URL para localhost caso queira apontar para o back-end local: src/services/baseUrl.js
